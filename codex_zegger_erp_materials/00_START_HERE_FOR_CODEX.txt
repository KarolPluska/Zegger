NOWE NAZEWNICTWO TECHNICZNE - KANON
-----------------------------------
Ustandaryzuj nowy system tak:

- Nazwa pluginu: Zegger ERP
- Folder pluginu: zegger-erp
- Główny slug/system slug: zegger-erp
- REST namespace: zegger-erp/v1
- PHP class prefix: ZERP_
- DB table prefix (po wp_): zerp_
- JS global namespace: window.ZEGGER_ERP

Jeżeli w implementacji musisz utrzymać kompatybilność legacy, możesz zostawić wybrane warstwy zgodności dla starych endpointów, ale nowy kod ma być już oparty o nowe nazewnictwo.
